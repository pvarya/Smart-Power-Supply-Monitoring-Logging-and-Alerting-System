Smart Power Supply Monitoring, Logging, and Alerting System

Introduction

The main feature of the project is, it can give us the server analysis at the real time so an user can choose or can swap the server for further use, if that server goes down.
The outcome of this project is that the internet user never face the server problem because the user come to a out the server condition instantly.
The outcome of this project as a student is, it improves our the coding on microcontroller and the ease of using microcontroller and how to analyse any real time data on computer which is very different part to learn.
